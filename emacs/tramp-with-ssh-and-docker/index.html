Emacs tramp with ssh and docker

Nov 15, 2015 • nix • ssh • tramp • emacs • docker •

TRAMP предоставляет возможность удаленной работы с файлами на различных устройствах,
как с локальными, по средствам различных протоколов. Рассмотрим некоторые из них …

Как использовать

C-x C-f /scp::user@server/home/user/file

Настройки в emacs

(use-package tramp ;; with use-package
  :config
  (setq-default tramp-default-method "scp")) ;; for performance

;; or default

(require 'tramp)
(setq-default tramp-default-method "scp")

;; use cache default
(setq remote-file-name-inhibit-cache nil) ;; if need disable

Настройки ssh

# ~/.ssh/config

Host *
     ControlMaster auto
     ControlPath ~/.ssh/sockets/%r@%h:%p # need mkdir sockets
     ControlPersist 4h

Host Example
     HostName example.com
     User example

Используем с ssh alias`ом

C-x C-f /scp::Example/home/user/file

Редактируем файл c sudo

Вызываем функцию для read-only файла …
НЕ забываем завершать tramp сессию от sudo, чтобы не было недоразумений …
Например: при сохранении *.el => byte-compile-current-buffer в *.elc c правами root …

(defun sudo-edit-current-file ()
  (interactive)
  (let ((my-file-name) ; fill this with the file to open
        (position))    ; if the file is already open save position
    (if (equal major-mode 'dired-mode) ; test if we are in dired-mode
        (progn
          (setq my-file-name (dired-get-file-for-visit))
          (find-alternate-file (prepare-tramp-sudo-string my-file-name)))
      (setq my-file-name (buffer-file-name)
            position (point))
      (find-alternate-file (prepare-tramp-sudo-string my-file-name))
      (goto-char position))))

(defun prepare-tramp-sudo-string (tempfile)
  (if (file-remote-p tempfile)
      (let ((vec (tramp-dissect-file-name tempfile)))

        (tramp-make-tramp-file-name
         "sudo"
         (tramp-file-name-user nil)
         (tramp-file-name-host vec)
         (tramp-file-name-localname vec)
         (format "ssh:%s@%s|"
                 (tramp-file-name-user vec)
                 (tramp-file-name-host vec))))
    (concat "/sudo:root@localhost:" tempfile)))

Редактируем файлы в docker container`е

Несмотря, что Emacs – старый зверь, ничего новое ему не чуждо …
Теперь для разработки не обязательно монтировать папку приложения в docker container,
а просто добавить (или git pull), отредактировать и результат закоммитить git repositor`ий …

(use-package docker
  :defer t
  :ensure t
  :config
  (docker-global-mode)) ;; manage docker containers

(push
 (cons
  "docker"
  '((tramp-login-program "docker")
    (tramp-login-args (("exec" "-it") ("%h") ("/bin/bash")))
    (tramp-remote-shell "/bin/sh")
    (tramp-remote-shell-args ("-i") ("-c"))))
 tramp-methods)

(defadvice tramp-completion-handle-file-name-all-completions
    (around dotemacs-completion-docker activate)
    "(tramp-completion-handle-file-name-all-completions \"\" \"/docker:\"
    returns a list of active Docker container names, followed by colons."
    (if (equal (ad-get-arg 1) "/docker:")
        (let* ((dockernames-raw
        (shell-command-to-string
        "docker ps | awk '$NF != \"NAMES\" { print $NF \":\" }'"))
            (dockernames (cl-remove-if-not
                         #'(lambda (dockerline) (string-match ":$" dockerline))
                         (split-string dockernames-raw "\n"))))
          (setq ad-return-value dockernames))
          ad-do-it))
C-x C-f /docker:container:/etc/passwd RET ;; simple use
